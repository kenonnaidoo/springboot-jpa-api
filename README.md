A basic Java web API created by Kenon Naidoo.

<b>Host: http://127.0.0.1:8080</b>

<b>Endpoint Features:</b>

<b>Get all users (GET)</b><br>
<b>/users/</b>

<b>Get user by id (GET)</b><br>
<b>/users/{id}</b>

<b>Create user (POST)</b><br>
<b>/users/</b><br>
<b>
{
  "firstName" : "Tony",
  "lastName" : "Stark"
}
</b>

<b>Update user (PUT)</b><br>
<b>/users/{id}</b><br>
<b>
{
  "firstName" : "Tony",

</b>

<b>/users/{id}</b><br>
